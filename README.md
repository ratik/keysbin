#KeysBin

Simple password manager that crypt data on client and store on the server in MongoDb.


#Installation 

```sh
$ git clone https://github.com/ratik/keysbin
$ cd keysbin
$ npm install
$ cd bower_components/openpgp
$ npm install
$ grunt
$ cd ../..
$ gulp 
$ node index
```
